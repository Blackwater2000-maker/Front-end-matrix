How to model it

Platform + implementation + variable media

Treat the software as a recurring infrastructure layer, then model direct mail as variable spend tied to campaign volume.

  • Fixed monthly software fee
  • One-time deployment fee
  • Variable mail volume cost
  • No seat-based pricing complexity

Commercial structure

The pricing model is intentionally simple so operator teams and executive buyers can understand exactly what is fixed versus volume-driven.

Rollout fit

This structure works best for dealer groups that want predictable platform cost and flexible direct-mail throughput as campaigns expand.
