How we're different

Built for execution, not just analysis.

Data unification is table stakes. These are the execution capabilities that change what a dealership can actually do.

Execution

Every campaign you should run but don't.

The team is already at capacity. AutoCDP isn't — six agents run around the clock. Every signal triggers a campaign, every customer gets contacted, no backlog.

Attribution

Direct mail with a paper trail.

AI-written, handwriting-simulated postcards with a unique QR code on every piece. When scanned, you know exactly who, when, and which campaign booked the appointment — per-piece, not modeled.

Closed-loop learning

Sharper with every campaign.

After each send, the Learning Ledger reconciles real outcomes against each creative variant. Winning angles get promoted, weak ones retired, and proven anchors feed the next campaign so results compound.

The intelligence layer

Six agents. One shared state.

Every agent sees every other agent's decisions as they happen — no handoffs, no lost information. The swarm runs as one coordinated system.

AGENT 01Opus 4.8

Orchestrator

Decomposes a plain-language goal into structured subtasks, routes work to each specialist, tracks shared state, handles retries, and decides when to re-run on new DMS data.

AGENT 02Sonnet 4.6

Data Intelligence

Computes per-customer churn probability, lifetime-value estimate, lifecycle stage, and channel fit — returning a ranked audience with confidence scores.

AGENT 03Sonnet 4.6

Audience Targeting

Applies channel-fit scoring from the global learnings database, assigning each customer a single channel or a sequenced multi-channel plan with conditional triggers.

AGENT 04Sonnet 4.6

Creative Generation

Writes unique content per recipient from their vehicle, mileage, and service history — postcard, SMS, and email. No template strings; a generation-time FCRA gate shapes every piece.

AGENT 05Haiku 4.5

Compliance Review

Runs before any send, validating each message against TCPA, CAN-SPAM, FCRA, and DNC. Any failure halts that recipient with a structured reason in sub-200ms.

AGENT 06Sonnet 4.6

Learning & Optimization

Reconciles delivery, QR-scan, call, and appointment events against each variant — driving promote/retire decisions and writing PII-stripped patterns to the network.
