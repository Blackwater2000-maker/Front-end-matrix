The intelligence layer

Six agents. One shared state.

Every agent sees every other decision as it happens. No handoffs. No lost context. No dead zones between planning and execution.

Agent 01Opus 4.8

Orchestrator Agent

Decomposes a plain-language goal into structured subtasks, routes work to specialists, tracks shared state, handles retries, and decides when to re-run.

Agent 02Sonnet 4.6

Data Intelligence Agent

Scores every customer on churn, value, lifecycle stage, and channel fit using RO history, recency, vehicle age, and behavioral signals.

Agent 03Sonnet 4.6

Audience Targeting Agent

Selects the optimal channel and timing per customer, then generates the final execution plan downstream agents act against.

Agent 04Sonnet 4.6

Creative Generation Agent

Writes personalized copy for postcard, SMS, and email using vehicle, mileage, service history, and current offer without mail-merge-style templates.

Agent 05Haiku 4.5

Compliance Review Agent

Enforces TCPA, CAN-SPAM, FCRA, and DNC rules before any send. Any blocked recipient receives a structured reason and never enters execution.

Agent 06Sonnet 4.6

Learning & Optimization Agent

Reconciles delivery, QR-scan, call, and appointment outcomes against creative variants and updates each rooftop’s Learning Ledger.
